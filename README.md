# Exo-Gamify-LM
ExoQuest/XQuest: Gamified Human-in-the-Loop Machine Learning for Exoplanet Detection. This end-to-end pipeline and mobile-first game (XQuest) streamlines vetting of TESS transit signals. Features ExoReg, a relational registry of K-Dwarf systems, using community intuition to train high-accuracy ML habitability models.
